# Application Tracker (Full Stack)

A full-stack application tracker built with **Django (backend)**, **React + Vite (frontend)**, and optional **Docker support**.

---

## Tech Stack

### Backend
- Django 4+
- Django Ninja (API layer)
- SQLite / PostgreSQL (configurable)
- Python 3.9+

### Frontend
- React (Vite)
- React Router
- Axios (API calls)
- Modern component-based UI

### DevOps
- Docker (optional)
- Docker Compose
